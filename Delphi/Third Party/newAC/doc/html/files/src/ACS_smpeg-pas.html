TMP3In

The mp3 file/stream decoder that uses the Windows built-in decoder, descends from TWMIn

TMpgIn

Yet another mp3 file/stream decoder that uses libmpg123.dll.  Descends from TAuTaggedFileIn.  This decoder provides better sound than the Windows one but doesn’t suppoer the precise positioning.

Reurns the file’s Id3v2Tags if available.  If Id3v2Tags are not present in the file, the Id3v1Tags values are returned by this proeprty.
