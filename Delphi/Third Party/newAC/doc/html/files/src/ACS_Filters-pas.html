TSincFilter

This component implements a windowed-sinc filter.  The filter can work in low-pass, high-pass, band-pass, band-reject, or all-pass mode.  A descendant of TAuConverter.

Properties

FilterType

Use this property to set the desired filter type: low-pass, high-pass, band-pass, or band-reject Curretnly only low-pass and high-pass filters are stable.

NumberOfPoles

Use this property to set the number of poles for the filter.  Allowed values are 2, 4, 6 ...  16.  Note that filters with high number of poles may become unstable in which case their operation results in floating pointoverflow.

HighFreq

Use this property to set the high cut-off frequency.  This property applies to high-pass, band-pass, and band-reject filters.  The value of this property should not be greater than the half of the samping rate.

LowFreq

Use this property to set the low cut-off frequency.  This property applies to low-pass, band-pass, and band-reject filters.  The value of this property should not be greater than the half of the samping rate.

Ripple

Use this property to set the band-pass ripple.  The reasonable values for Chebyshev filters are are 0.30.5.  If you set this property to zero the filter’s response becomes that of Butterworth filter.
