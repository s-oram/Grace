This folder contains demos showing how to convert between audio file formats.
For the formats supporting tags (all, except wav and, currently, Monkey Audio) you can edit tag information.
Following is the short description of each demo:

 MP3Converter - converts mp3 files to wav format.

 MP32WMA - converts mp3 files to wma format preserving tags information.

 Wav2Ape - converts wav files to Monkey's Audio

 Wav2FLAC - converts wav files to FLAC

 Wav2MPC - converts wav files to Musepack

 Wav2Ogg - converts wav files to Ogg Vorbis

 Wav2Ogg-2 - converts wav files to Ogg Vorbis with a different set of controls

 Wav2WMA - converts wav files to Windows Media Audio (WMA)

 Wav2WMA-2 - converts wav files to Windows Media Audio using direct codec/format selection.

 Wav2WV - converts wav files to WavPack

 WMA2Ogg - This converter demostrates how to select WMA output formats
