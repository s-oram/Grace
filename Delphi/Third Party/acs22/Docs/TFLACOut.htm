TFLACOut

Properties Methods Events

Unit: ACS_FLAC

Descends from: TACSFileOut

TFLACOut component stores incoming audio stream in a FLAC lossless audio compression format.

Note that in order to use this component you will need libFLAC.dll under Windows, or libFLAC library under Linux.

See also: TFLACIn
