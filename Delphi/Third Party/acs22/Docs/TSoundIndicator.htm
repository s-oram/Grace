TSoundIndicator

Properties Methods Events

Unit: ACS_Indicator

Descends from: TACSConverter

TSoundIndicator represents audio information in the form of freqency spectrum (by applying FFT to the incoming data). In order to use this component you have to put it into the audio chain between input and output components.

It is important to note that TSoundIndicator is a non-visual component. It just provides data for graphical representation. OggPlayer demo shows how this copmponent may be used.
