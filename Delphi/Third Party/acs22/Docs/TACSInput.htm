TACSInput

Properties Methods Events

Unit: ACS_Classes

Descends from: TComponent

TACSInput is the base class for all input and converter components. It introduces such properties as BitsPerSample, Channels, SampleRate, Size and such methods as Flush, Init, GetData (these methods are abstract and should be overridden in the descendant classes.

See also: TACSFileIn, TACSOutput
