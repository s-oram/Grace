TMP3ToWav Events

OnDone : TMADDoneEvent;

TMADDoneEvent = procedure(Sender : TComponent; Success : Boolean) of object;

This event is invoked when the component has finished the conversion process. Success indicates whether or not the conversion was successful.

OnProgress : TMADProgressEvent;

TMADProgressEvent = procedure(Sender : TComponent) of object;

This event is invoked every time the component's Progress property value changes.
