TMACIn

Properties Methods Events

Unit: ACS_MAC

Windows only

Descends from: TACSFileIn

TMACIn component reads audio data from Monkey Audio Compression format (this files usually have an ".ape" extension). Monkey Audio is a lossless compression format that reduces audio data to ~ 1/3 of its original size. In spite of being a descendant of TACSFileIn and so of TACSStreamedInput, TMACIn doesn't allow the use of generic streams (the Stream property is ignored in this component). You will need MACDll.dll file in order to use the MAC codec.

See also: TMacOut
