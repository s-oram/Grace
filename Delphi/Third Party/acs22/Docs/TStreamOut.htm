TStreamOut

Properties Methods Events

Unit: ACS_Streams

Descends from: TACSStreamedOutput

TStreamOut component can save raw audio data to any TStream descendant assigned to its Stream property. Note that TStreamOut doesn't store such audio stream parameters as sample rate or sample size so you have to store these parameters somewhere else if you need them.

See also: TStreamIn
