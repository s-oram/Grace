TACSFileIn

Properties Methods Events

Unit: ACS_Classes

Descends from: TACSStreamedInput

TACSFileIn is the base class for all input components that read data from files (or from other streams in the corresponding file format). It introduces such properties as FileName, StartSample, EndSample, Loop, and Valid and Jump, Seek, SetStartTime, SetEndTime methods.
