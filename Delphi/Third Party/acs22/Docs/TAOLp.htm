TAOLive Properties

DefaultDriver : String;

Readonly

Read this property to obtain the name of the default libao driver. This is the driver that is used if Driver property is set to ''.

Driver : String;

Use this property to set the driver for libao playback. The driver's name must be the one from the Drivers list. If this Driver is set to '', the default libao driver is used.

See also: IsDevicePlayable

Drivers : TStringList;

Readonly

Read Drivers to get the list of live drivers installed with libao.

See also: IsDevicePlayable

Volume : Integer;

Use this property to set audio volume. The maximum value is 255 (default), the minimum - 0.
