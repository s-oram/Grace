TALSAAudioOut

Properties Methods Events

Unit: ACS_ALSA

Descends from: TACSOutput

TAudioOut component allows audio stream to be played via using ALSA drivers directly. Note that TAudioOut component can be used with all drivers (including ALSA) so using TAudioOut is more preferable (and recommended). Use TALSAAudioOut only if you write ALSA-specific applications (for example those that use ALSA device-naming conventions).

Note that in order to use this component libasound.so library should be installed.

See also: TALSAAudioIn
