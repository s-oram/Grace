TMACOut

Properties Methods Events

Windows only

Unit: ACS_MAC

Descends from: TACSFileOut

TMACOut component stores incoming audio stream in a Monkey Audio Compression format (this files usually have an ".ape" extension). Monkey Audio is a lossless compression format that reduces audio data to ~ 1/3 of its original size. Unlike other file output components, TMACOut doesn't allow the use of generic streams for outuput.

You will need MACDll.dll file in order to use the MAC codec.

See also: TMACIn
