TBWFilter

Properties Methods Events

Unit: ACS_Filters

Descends from: TACSConverter

TBWFilter component is an implementation of a simple Butterworth filter. This component takes in an audio stream, performs band-pass/band-reject operations on it and passes the resulting stream to some other component. Note that TBWFilter operates on 16-bit streams only.
