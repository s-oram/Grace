TACSOutput

Properties Methods Events

Unit: ACS_Classes

Descends from: TComponent

TACSOutput is the base class for all ACS output components. It introduces such properties as Input, Progress, Status, SuspendWhenIdle, ThreadPriority, and TimeElapsed. The methids introduced in this class are Pause, Resume, Run, Stop. TACSOutput also introduces two events: OmDone and OnProgress.
